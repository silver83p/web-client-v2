Backup Account

This will download a file that contains your account data. You can use this file to restore your account on another device.

If you are signed in, this will only save the current account. If you are not signed in, this will save all accounts.

Remove Account

Before removing your account, please export your account data to ensure you have a backup. This action only removes the account from this device and not from the network. You can use the account again after restoring it from the backup file.
